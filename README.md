Telegram bot for ISHITR task

## Telegram-bot access

You can try it yourself by:
t.me/test_isgit_bot
or
@test_isgit_bot

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
